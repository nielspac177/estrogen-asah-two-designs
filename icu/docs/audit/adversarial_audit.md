# Adversarial methodological audit (7-agent panel)

Independent adversarial reviewers, one per methodological dimension, followed by a synthesis. Generated for the estrogen→DCI null. This is the record referenced by ADR-0005.

## Overall verdict

The null is credible and should stand as reported. The adjusted OR of 0.86 (0.58-1.28) is NS, the crude gradient runs OPPOSITE to the a-priori hypothesis in every operationalization (vasospasm-only crude OR 0.39, procedure 0.62, infarction 0.76, composite 0.59), the E-value is only 1.54, and the sources are directionally discordant (eICU 1.56 vs MIMIC 0.73) with a k=2 random-effects pool of 0.89 (0.46-1.73) that is essentially uninformative. Critically, the competing-risk/survival-bias story is NOT the dominant issue and should not be promoted to the lead explanation. The dominant, near-fatal problem is NON-IDENTIFIABILITY: the exposure is a deterministic dichotomy of age (post = 1[age>=51], derive.py:11,21), so within women there is zero age overlap across strata (structural positivity violation). The primary model then conditions on continuous age (models.py:19,78), regressing DCI on a step function of age net of a linear function of the SAME age, which (a) contradicts the project's own IPW code that drops age for exactly this reason (weighting.py:16-20), (b) makes the 'post' coefficient a functional-form artifact rather than a menopause contrast, and (c) yields a false-precision CI. The dominant open confounding path is age's DIRECT effect on vasospasm (younger aSAH patients reliably have MORE vasospasm), which is collinear with the exposure and points the wrong way for the hypothesis; it is larger than the E-value and cannot be adjusted away. Layered on top is differential ascertainment: the composite is not one construct across sources (3.3% eICU vs 33.1% MIMIC; delayed-infarction limb is 0/1227 in eICU vs 18.9% in MIMIC and uses no delay window, violating Vergouwen 2010; ICD-9 has no vasospasm code so pre-ICD-10 records are structural false negatives), and surveillance intensity falls with age, plausibly manufacturing the pre>post vasospasm gradient. Competing mortality (14.8% vs 7.7%) is real and plausibly depresses postmenopausal DCI ascertainment, but with no time-to-DCI it is unquantifiable, unsignable, and partly a measurement artifact (fatal DCI miscoded as death) that no Fine-Gray/cause-specific model can repair; it is a secondary, exploratory concern, not the explanation. Bottom line: this cross-sectional coded pilot cannot separate estrogen from ageing or from ascertainment, and every 'correction' that would produce the hypothesized OR>1 does so mechanically or by opening a collider. Report as null/hypothesis-generating; do not fish.

## Most likely LEGITIMATE positive (endorsed design)

The age x sex restricted-cubic-spline difference-in-differences (men as the chronological-ageing reference; menopause estimand = female-minus-male age effect at 55 vs 45). It is the single analysis most likely to yield a LEGITIMATE literature-consistent positive because it is the only design that structurally confronts the fatal problem, age-menopause collinearity. By borrowing men, who age without an abrupt estrogen transition, it subtracts the shared ageing gradient (the dominant confounder that otherwise makes 'post' non-identifiable and drives DCI down with age), so a residual female-specific UPWARD inflection in DCI odds after ~50 would be genuinely attributable to something estrogen-like rather than to ageing, and it does not require conditioning on age within an age-defined exposure, adjusting for mortality (collider), or re-coding death as an outcome (mechanical inflation). It is fully prespecifiable as confirmatory (fix knots, df, contrast ages, estimator, source handling), and it can be paired with the competing-risk sensitivities and a surveillance-utilization proxy to rule out the two rival explanations. The honest caveat: it is predicted to come back flat/null (the ascertainment and ageing gradients likely dominate), and even a positive rests on the untestable assumption that ageing acts identically on DCI in both sexes absent menopause; but if any route in this dataset produces a defensible OR>1, this is the one, whereas the death-inclusive composite and the eICU-only estimate would produce OR>1 only as artifacts. Note too that the cited human evidence (NHS) links low endogenous estrogen to aSAH INCIDENCE, not DCI severity, so even a clean positive would be a novel extension, not a confirmation of established literature.

## Pre-specified analysis plan

Register a SAP and lock it before any further model fitting; version-control it with a pre-data timestamp (note: ADR-0003 is dated 2026-07-01, same day as all outputs, so nothing is currently pre-locked). 1) TARGET ESTIMAND, stated explicitly: because menopausal state is a deterministic function of age here, the estrogen-specific controlled direct effect is NOT identifiable; the confirmatory question is reframed as the female-specific deviation in DCI odds across the menopausal transition NET of the shared ageing gradient (age x sex spline difference-in-differences, men as ageing reference), with a co-primary positivity-respecting peri-menopausal-band contrast. 2) PRIMARY MODEL: whole-cohort logistic, age as restricted cubic spline (knots fixed at 10/50/90th percentiles) interacted with sex; report the female-minus-male age contrast at prespecified ages (55 vs 45). Never place both a smooth age term and a menopause step in one model; never adjust age out of an age-defined exposure. 3) OUTCOME locked as coded cerebral vasospasm unioned with rescue endovascular therapy (the two components ascertained in BOTH sources), ICD-10 era only; delayed infarction is demoted to a MIMIC-only sensitivity that applies a real >48-72h post-ictus window excluding procedure/rebleed-attributable infarcts (Vergouwen 2010). 4) SOURCES reported STRATIFIED (eICU, MIMIC) as primary; no single pooled OR given the ~10x ascertainment gap and k=2. 5) ESTIMATOR: Firth penalized likelihood for sparse cells; wild-cluster bootstrap for eICU inference; do NOT use hospital CRVE (degenerate: MIMIC is one cluster carrying ~81% of events); fix the broken overlap-weight estimator (freq_weights coerces fractional weights to integer counts -> invalid SEs). 6) MENOPAUSE CUTOFF fixed at 51 with NO sweep; a single co-primary fuzzy-RDD narrow-window contrast (women 45-55) declared in advance. 7) COMPETING RISK handled by prespecified death-inclusive composite AND day-3 landmark AND Manski bounds, reported together as sensitivities with early/late death decomposition; survivor-restriction and adjusting for mortality/LOS are PROHIBITED (they open the Severity->Death<-DCI collider). 8) MULTIPLICITY: one primary test at alpha=0.05; a prespecified specification curve across all defensible forks with FDR across the sensitivity family; report median OR, position of the primary, and fraction/direction of significant forks. 9) A positive is credible only if the mechanism-predicted OR>1 (a) survives in the pooled/source-stratified primary, NOT only in a selected fork, (b) holds in BOTH sources, (c) survives the competing-risk sensitivities without relying solely on death-recoding, and (d) is accompanied by a measured surveillance/angiography-utilization proxy showing the signal is not ascertainment-driven. Honest note: a genuinely credible confirmatory test requires a redesigned study with time-resolved DCI ascertainment, admission-grade (Hunt-Hess/WFNS) adjustment, and an estrogen lever that varies independently of age.

## P-hacking warnings

- Source selection: elevating the eICU arm (OR 1.56, hypothesis-consistent) over MIMIC (0.73) or the null pool (0.89) is the single most dangerous available fork; the eICU direction is exactly what differential coding produces at a ~3% ascertainment rate.
- Menopause-cutoff sweep: age_cut is a free argument (default 51); silently re-cutting at 45/50/52/55 moves the OR across 1.0 and is pure fishing if used to reach significance.
- age_c toggle: including vs dropping continuous age changes both the estimate AND its interpretability (models.py keeps age_c; weighting.py drops it), an undisclosed researcher choice; adjusting age out of an age-defined exposure yields a false-precision CI (0.58-1.28) that excludes functional-form uncertainty.
- Competing-risk narrative as unfalsifiable rescue: with no time-to-DCI, the survival-bias story cannot be tested and every version of it (death-inclusive composite, landmark exclusion) mechanically pushes the estimate UPWARD toward the hypothesis, a predictable, one-directional garden-of-forking-paths signature and HARKing-adjacent reinterpretation of a null.
- Outcome-definition shopping: the OR swings 0.39->0.59->0.76->0.86 across composite/component choices; the composite's larger event count is power to detect coding artifacts, not the menopause contrast, and the invalid delayed-infarction limb (no delay window, 100% source-confounded, contributes 80 MIMIC-only events) inflates it.
- Survivor restriction / adjusting for mortality or LOS to 'see through' competing risk opens the Severity->Death<-DCI collider and will manufacture a spurious protective (or, if inverted, harmful) estrogen effect.
- HRT (OR 3.39) presented as causal: confounded by indication and immortal-time biased; a low p-value here is meaningless.
- Random-effects pooling with k=2 sources (tau^2 from two studies) launders real ascertainment heterogeneity into apparent sampling error.
- Multiple inference frameworks (cluster-strong logistic, RE meta, IPW, E-value) for one estimand let the analyst foreground the most favorable framing.
- Estimator degeneracy passed off as inference: hospital CRVE with one MIMIC cluster carrying ~81% of events, and overlap weights fed to statsmodels freq_weights as integer counts, both return invalid/too-narrow CIs.
- Selecting men-vs-women (OR 1.13) as evidence on the estrogen question: it confounds sex with age and the whole ascertainment structure and cannot arbitrate the hypothesis.
- No pre-registration and no multiplicity control currently exist (>100 defensible ORs, zero alpha adjustment); with a stated goal of 'finding a positive,' assume any positive is a false positive until a locked multiverse proves otherwise.

## Per-dimension critiques

### Competing risks & survival / immortal-time bias
*Proposed analysis:* Do not pretend to fit Fine-Gray/cause-specific Cox on DCI (no time-to-DCI exists). The defensible, runnable substitute is a three-part package reported together, since time-to-DEATH IS available (LOS / admit-disch offsets): (1) a competing-event composite endpoint 'DCI OR in-hospital death' re-fit with the same adjusted cluster-strong logistic, counting death as an event directly resolves the masking without needing event times; (2) a day-3 landmark logistic excluding patients who died within ~72h of admission (i.e., before the vasospasm window), removing early competing deaths from both arms; (3) Manski/tipping-point bounds that assign the unobserved DCI status of early decedents at both extremes (all-DCI vs no-DCI) to show how sensitive the direction is. Add presentation-severity adjustment (APACHE in the eICU subset) as a sensitivity. Interpret concordant upward movement across all three as weak corroboration, never confirmation.
*Predicted direction:* Prior literature/mechanism (17b-estradiol protective in ovariectomised SAH models; ERb/GPER1 on cerebral endothelium; early menopause / short reproductive span -> higher aSAH risk in NHS) predicts low-estrogen postmenopausal state -> MORE DCI, i.e., the desired OR is >1. Every element of a competing-risk correction here mechanically pushes the postmenopausal estimate UPWARD: counting death as an event adds the postmenopausal excess deaths as 'events,' and landmark-excluding early deaths removes disadvantaged postmenopausal denominator mass. So the reanalysis is predicted to move the estimate from 0.86 toward/above 1, precisely the hypothesized direction, which is why a 'positive' result would be intrinsically suspect rather than confirmatory.
*p-hack risk:* high · *prespecifiable:* False
*Flaws:*
- The primary model (src/estrogen_asah_dci/analysis/models.py, primary_logistic) is a binary logistic on a coded DCI composite that ignores competing death entirely: it implicitly assumes premenopausal and postmenopausal women had equal opportunity to develop and be coded for DCI, despite a ~2x in-hospital mortality gap (14.8% vs 7.7%). The competing-risk concern is invoked in prose (manuscript.md lines 34, 133, 157) but never entered into any estimator.
- Death is NOT a clean competing risk here: delayed cerebral infarction is itself a component of the DCI composite AND a leading cause of death after aSAH. So death is partly the outcome, not merely a competing event. In the ICD-9 era (author-acknowledged insensitive coding), fatal DCI is plausibly recorded as 'death' with no vasospasm/infarct code, producing DIFFERENTIAL outcome misclassification in the higher-mortality postmenopausal arm. This is a measurement/ascertainment problem that no competing-risk model (Fine-Gray or cause-specific) can fix.
- There is no time-to-DCI in the data (author-acknowledged; cohort carries only icu_los_days, a death flag, admit/disch times). Therefore the 'principled' competing-risk estimators (Fine-Gray subdistribution hazard, cause-specific Cox) are literally not estimable for DCI, they require the event time that does not exist. The manuscript asserts a competing-risk explanation it cannot quantify or even sign.
- Competing risk and immortal-time bias are conflated. Immortal-time bias genuinely applies only to the time-varying in-hospital HRT exposure (OR 3.39), you must survive long enough to be dosed. Menopausal state is fixed at admission, so the primary contrast has NO immortal time. The write-up muddies which bias attaches to which exposure.
- The excess absolute postmenopausal mortality is only ~7 percentage points, and its TIMING (early ictal/rebleed death before the day 4-14 vasospasm window vs late DCI-related death) is never decomposed. Only early, pre-window deaths can mask DCI ascertainment; late deaths cannot. Without splitting early vs late death, the claim that competing mortality 'plausibly depresses' postmenopausal DCI is unquantified and could be negligible.
- Presentation severity (Hunt-Hess/WFNS grade; APACHE available in eICU only) drives BOTH early death and DCI and is uncontrolled in the primary model (covariates are only age_c, htn, smoking, diabetes). Any competing-risk or landmark reanalysis remains confounded by admission severity, so a direction reversal could not be attributed to estrogen.
- The reported E-value (1.54) and the entire confounding discussion address unmeasured CONFOUNDING; they say nothing about the selection/competing-event structure. The paper offers no bound whatsoever on the survival/competing-risk bias it repeatedly names.

### Confounding structure & estimand (implicit DAG, age–menopause collinearity, collider control, target estimand)

Among WOMEN only, fit DCI ~ f(age) with age continuous (restricted cubic spline, or linear as a sensitivity), logistic with cluster-strong SE by hospital, and NEVER include both a smooth age term and the post indicator. Then formally test whether adding a menopause threshold indicator at 51 improves fit over the smooth age model (likelihood-ratio / regression-discontinuity-style test for a step at the cutoff). Report the age slope as the primary quantity and state the estimand explicitly as 'association of chronological age with coded DCI in women', conceding that endogenous estrogen cannot be isolated from age in this design. Pre-specify that the estrogen-specific question is answerable ONLY via an estrogen lever that varies independently of age, here that is exposure HRT, which is confounded by indication (OR 3.39), so it is reported as hypothesis-generating with a negative-control outcome and E-value/bounding, not as an adjusted effect. Where severity exists (eICU), report a severity-adjusted estimate separately rather than pooling eICU and MIMIC into one exchangeable effect. Handle death as a competing risk (cause-specific or subdistribution framing, or a death-inclusive composite), never by conditioning on survivors or adjusting for mortality/LOS.
*Predicted direction:* Clinical and epidemiologic priors: younger aSAH patients reliably develop MORE cerebral vasospasm/DCI, so continuous age will show a NEGATIVE age–DCI slope (DCI falls with age), and the menopause-threshold term will be null once smooth age is in the model. ### Exposure operationalization: isolating the menopause contrast from chronological ageing
*Proposed analysis:* Fit the WHOLE cohort (men + women) with age modeled flexibly as a restricted cubic spline (pre-specified 3-4 knots at fixed percentiles, e.g. 10/50/90) interacted with sex, using men as the chronological-ageing reference. Define the menopause estimand as a difference-in-differences: the female-specific excess in DCI log-odds across the menopausal transition (the female-minus-male age effect evaluated at, e.g., 55 vs 45), which nets out the shared ageing gradient that men and women both experience. Report this alongside a single pre-specified fuzzy-RDD-style narrow-window sensitivity analysis (women 45-55, pre 45-50 vs post 51-55) that acknowledges cutpoint misclassification, and a pre-specified competing-risk (cause-specific or Fine-Gray) model given postmenopausal women die more. Do NOT re-cut the age threshold or window post hoc. This beats the age-51 dichotomy because (a) it models age flexibly instead of assuming linearity, and (b) it borrows men to subtract pure ageing - the dichotomy, restricted window, and 3-level sex coding cannot do either.
*Predicted direction:* Prior literature (17b-estradiol protective in ovariectomised SAH models; ERb/GPER1 in cerebral endothelium; NHS early-menopause -> higher aSAH risk) predicts lower estrogen (postmenopausal) -> MORE DCI, i.e. a female-specific upward inflection in DCI odds after ~50 relative to the male age trend. However, the actual crude data run the OPPOSITE way (post 10.4% vs pre 16.3%; adjusted OR 0.86) and DCI risk is known epidemiologically to fall with age. Because older/postmenopausal women die more (competing risk) and receive less aggressive angiographic vasospasm workup, and ICD-9-era DCI coding is insensitive, I predict the flexible age x sex spline will show a FLAT-to-DECLINING female DCI-age slope with no menopause inflection - the ascertainment/survival gradient will dominate and mask or reverse any true estrogen effect.

*Flaws:*

Because `post` is a deterministic threshold of age, the `post` coefficient is identified only by the discontinuity at 51 conditional on a linear age trend. If the true age-DCI relationship is nonlinear (it almost certainly is - DCI/vasospasm risk is well known to be higher in younger aSAH patients and to decline monotonically-but-curved with age), the linear age term is misspecified and the step at 51 absorbs the residual curvature. OR 0.86 is therefore not cleanly a 'menopause' effect; it is partly a curvature-absorption artifact of forcing a linear age adjustment. Adjusting for age linearly does NOT 'control for ageing' when the exposure is itself a function of age.
- A single population-median cutpoint (MENOPAUSE_AGE=51) maximizes exposure misclassification exactly at the boundary that dominates the contrast. Individual age at natural menopause spans ~45-56; a large fraction of women aged 48-51 are already peri/postmenopausal and some 51-54 still cycle, and surgical menopause (hysterectomy/oophorectomy, common and unmeasured here) makes many younger women postmenopausal. The 'premenopausal' (<51) stratum is thus contaminated with postmenopausal women and vice versa, biasing the OR toward the null - fully consistent with the observed 0.86 (0.58-1.28) NS.
- `age_cut` is a free function argument (default 51) and `menopausal_stratum`/`add_features` accept it directly, so the entire result can be silently re-cut at 50/51/52/etc. Combined with the choice of window width, spline df, and knot placement, this is a large undeclared specification search - a p-hacking surface, not a fixed design.
- Men - who age chronologically without menopause and are the only available 'pure ageing' control - are excluded from the primary and used only as a separate secondary sex contrast. The one quasi-identifiable menopause signal (a female-specific deviation from the male age gradient near 50) is discarded in the primary specification.
- `age_c` is centered on the full-cohort mean (including men) yet used in a women-only model, and, more importantly, age is capped/shifted at the top (MIMIC-IV sets >89 to 91; eICU bins '>89'). The continuous age adjustment is therefore corrupted precisely in the tail where postmenopausal women concentrate, so 'adjustment for ageing' is done on a censored covariate.
- The study reports the OR as a menopause effect while itself acknowledging age and menopausal state are collinear by construction. Restricting to women and adjusting for age cannot separate 'being postmenopausal' from 'being older'; the estimand actually reported is a threshold-at-51 jump net of a linear age trend, a quantity with no clean causal interpretation.

### Estimator & positivity/overlap
*Proposed analysis:* Reframe the primary contrast as a LOCAL, positivity-respecting comparison near the menopause threshold (a donut/fuzzy-RD-style design) and estimate it with a Firth penalized-likelihood logistic. Concretely: restrict to a pre-specified peri-menopausal band (e.g., 47-55, chosen a priori), regress DCI on post plus htn/smoking/diabetes and a WITHIN-BAND linear age term (age now has genuine overlap across both strata, so this adjusts for age without extrapolation), and fit with Firth's penalized likelihood using profile-penalized-likelihood confidence intervals rather than Wald/CRVE. Report this SEPARATELY for MIMIC-IV and eICU rather than pooling, because outcome ascertainment differs by an order of magnitude between sources; do not use hospital cluster-strong SEs (degenerate here), for eICU use a wild-cluster bootstrap or simply report the source-specific Firth estimate with its penalized CI. Present this as the identified estimand and relegate the age-adjusted full-range logistic to a clearly-labeled 'extrapolation, not identified' footnote. If a marginal/collapsible effect is wanted, g-computation is acceptable ONLY within the age band (where overlap holds), never on the full range.
*Predicted direction:* Prior literature (17b-estradiol protective in animal SAH; lower lifetime estrogen -> higher aSAH risk in NHS) predicts postmenopausal -> MORE DCI, i.e., OR>1. The crude and within-band data show the OPPOSITE: full-cohort post 10.4% vs pre 16.3%, and within the 45-55 band 23/136 (16.9%) pre vs 17/150 (11.3%) post (RR<1). A positivity-respecting local Firth model, having removed the age confounding that could only ever push toward the hypothesis via the survival-bias story, will most defensibly return a null or protective-direction estimate with a CI spanning 1 (the E-value is only 1.54 and the CI bound is already 1.0). The hypothesized OR>1 is expected to appear only in specific configurations that lean on the unmodeled competing-mortality argument.
*p-hack risk:* high · *prespecifiable:* True
*Flaws:*
- STRUCTURAL (deterministic) positivity violation that no estimator can fix. The exposure is defined as post = 1[age>=51] (features/derive.py MENOPAUSE_AGE=51; models.py:80). I confirmed in the cohort that pre-menopausal age support is 19-50 and post-menopausal is 51-91 with ZERO overlap. So Pr(post | age) is exactly 0 or 1 everywhere. The 'menopause' contrast is not nonparametrically identified; it is only an age extrapolation.
- The PRIMARY model conditions on age anyway (DEFAULT_COVARIATES includes 'age_c', models.py:19), which is internally inconsistent with weighting.py:16-19 that explicitly DROPS age because 'menopausal stratum is defined by an age threshold, so age is collinear with the exposure (no overlap).' You cannot have it both ways: with age_c in the model the 'post' coefficient is identified solely by the assumed linear functional form of age extrapolated across a gap with no common support; the reported CI (0.58-1.28) is false precision because it excludes functional-form uncertainty. If you instead drop age (as the IPW code does), you leave gross residual confounding by age (older aSAH = worse grade, more comorbidity, different aneurysm/rupture patterns).
- The proposed 'fancy' estimators do not rescue identification. AIPW/IPW/overlap weights/g-computation all require positivity. With age in the propensity model, PS(post|age) is 0/1, so IPW weights are undefined/explosive and overlap weights collapse to ~0. The overlap_weighted() already in strong.py:58-77 imports DEFAULT_COVARIATES (which contains age_c), so its ATO weights will degenerate; worse, it passes fractional weights via statsmodels freq_weights (strong.py:69), which treats them as integer counts and returns invalid, too-narrow standard errors. That estimator is broken as written.
- Degenerate cluster structure for the cluster-strong variance. I confirmed the women-only analytic set has 66 hospital clusters, but 65 are eICU and 1 is MIMIC-IV, and DCI events are massively concentrated: MIMIC contributes 108/133 events (81%) from its single cluster (30% event rate) vs eICU 25/748 (~3.3%). CRVE 'by hospital' assumes many exchangeable clusters of comparable influence; here one cluster drives the point estimate and the between-cluster variance is estimated off mostly-zero-event eICU sites. The pooled cluster-strong CI is unreliable and the pooled OR is essentially just the MIMIC OR (0.73).
- Small cells / separation once positivity is restored. The only credible fix (peri-menopausal age band, strong.py:38) shrinks the sample: band 45-55 has n=286, 40 events; split by source it is eICU 191/8 events across 65 hospitals and MIMIC 95/32 events in 1 hospital. A 4-covariate MLE logistic on 8 events (eICU band) is guaranteed to separate/be unstable; the by-source eICU full-cohort fit already shows this (OR 1.56, CI 0.44-5.61). Standard MLE logit (smf.logit, models.py:58) is the wrong estimator under this sparsity.
- Random-effects pooling with k=2 sources (meta.py via run_20) estimates tau^2 from two studies (reported tau2=0.0057), which is uninformative; the between-source discordance (eICU OR 1.56 vs MIMIC 0.73) reflects differential DCI ascertainment/coding density, not sampling error, so pooling them into one number is not defensible regardless of estimator.

### Multiplicity & researcher degrees of freedom (the garden of forking paths)
*Proposed analysis:* Replace the menu of estimates with a single locked confirmatory primary PLUS a full specification-curve (multiverse) analysis. (1) Pre-register ONE primary estimand: post-vs-pre DCI-composite logistic, age-51 cutoff, a fixed covariate set that EXCLUDES age (resolve the age_c/collinearity contradiction), pooled across sources with source as a stratification / random-intercept term rather than source-selected, tested once at alpha=0.05. (2) Demote everything else (3 composite variants, sex contrasts, HRT, source-specific, cutoff sensitivity, adjustment-set variants) to explicitly-labeled sensitivity/exploratory and enumerate the full grid, fitting the OR across ALL defensible forks (cutoffs x covariate sets x composite variants x source subsets). Plot the specification curve, report where the primary 0.86 sits in that distribution, the median OR, and the fraction of specifications with p<0.05 and their direction split. Apply FDR/Bonferroni across the pre-listed sensitivity family so the composite-variant set is read as robustness, not as extra chances at significance. This directly quantifies and neutralizes the forking-path problem instead of hiding it.
*Predicted direction:* Prior literature (17b-estradiol vasoprotection in ovariectomised SAH models; ERb/GPER1 expression; NHS reproductive-timing cohort) predicts postmenopausal -> MORE DCI, i.e., OR>1. The eICU arm (1.56) matches this and is the estimate a confirmation-seeking analyst would elevate. The honest prediction from a properly pooled, multiplicity-controlled multiverse is a specification curve centered near OR=1 (pooled 0.86-0.89, discordant sources tau2~0.006, E-value only 1.54), with a minority of significant forks split in direction rather than a coherent mechanism-predicted shift. In this dataset a mechanism-consistent positive is therefore expected only as a cherry-picked fork, not as a strong central tendency.
*p-hack risk:* high · *prespecifiable:* True
*Flaws:*
- Source-selection is the single most dangerous fork and it is already live in the pipeline. by_source() produces eICU OR=1.56 (in the hypothesized post->more-DCI direction) vs MIMIC OR=0.73 (opposite). An analyst told to 'find a positive' reports the eICU arm as the headline. Reporting a source-specific estimate as confirmatory when the pooled estimate is null (0.86 / 0.89) is a textbook forking-path artifact, not a finding.
- No pre-registered SAP and no multiplicity control anywhere. grep finds zero alpha/Bonferroni/Holm/FDR/pre-registration language; results.json emits raw p-values only. The one 'pre-specified' item (ADR-0003 composite variants) is a post-project ADR dated the same day (2026-07-01) as everything else, so nothing was locked before seeing data.
- The space of defensible specifications is enormous. Exposure contrasts: post-vs-pre, M-vs-F, M-vs-pre, M-vs-post, HRT (~5). Outcome operationalizations: composite + 3 pre-listed DCI variants (vasospasm-only, rescue-only, exclude-infarction) + mortality + poor disposition + ICU LOS (~7). Adjustment sets: base / +age / +modality / +APACHE / IPW (~4-5). Source subsets: pooled / eICU / MIMIC (3). Menopause cutoff: 45/50/51/52/55 (~5). The product is >100 defensible ORs from one dataset; at nominal 0.05 you expect several 'significant' hits by chance with no correction applied.
- The menopause cutoff (age 51) is a continuous variable dichotomized at an investigator-chosen threshold. 51 is defensible (median US menopause) but nothing in code or docs locks it, and shifting it 45<->55 is a silent knob that moves the OR. A cutoff sweep is unreported and would be pure fishing if used to find significance.
- Age adjustment is both a modeling error and a hidden fork. DEFAULT_COVARIATES in models.py includes 'age_c', so primary_logistic adjusts the post-vs-pre contrast for continuous age even though 'post' is a deterministic threshold function of age within women (near-collinear; you partial out the exposure). This contradicts the Methods claim that age was not used, and toggling age_c in/out is an undisclosed researcher choice that changes the estimate and its interpretability.
- The competing-risk / survival-bias narrative is an unfalsifiable post-hoc rescue. The crude data point the WRONG way (premenopausal 16.3% > postmenopausal 10.4%), and the higher postmenopausal mortality (14.8% vs 7.7%) is invoked to argue the true protective effect is masked. With no time-to-DCI this cannot be tested in the data; it functions as a HARKing-adjacent device that lets a null/opposite result be reinterpreted as 'consistent with the a-priori hypothesis.' Any downstream analysis motivated by it (e.g., competing-risk models, mortality-conditioned subsets) is exploratory, not confirmatory.
- Multiple inference frameworks are reported for one estimand (cluster-strong logistic, random-effects meta, IPW, E-value), each a separate degree of freedom in how the same post-vs-pre effect is presented; the analyst can foreground whichever framing is most favorable.
